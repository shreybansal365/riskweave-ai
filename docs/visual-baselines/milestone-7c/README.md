# Milestone 7C visual evidence

This directory contains the bounded Chromium release-candidate shortlist requested for Milestone 7C.
The captures use the local Docker application, the deterministic baseline plus all three showcase
scenarios, the synthetic administrator role, reduced motion, and the approved 1440×900 or 1024×768
viewport named in each filename.

The eight route captures and one contact sheet are additive evidence. They do not replace or modify
the historical Milestone 6, 7B, or brand baselines. In particular, the Overview captures must show
18 current persisted incidents separately from the 15 incidents represented by the exact 14-day
trend series.
